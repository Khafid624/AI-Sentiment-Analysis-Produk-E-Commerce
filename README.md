# AI-Sentiment-Analysis-Produk-E-Commerce
Aplikasi Analisis Sentimen AI berbasis Streamlit untuk mengklasifikasikan review produk e-commerce menjadi positif, negatif, atau netral. Menggunakan Python, NLP, dan machine learning dengan fitur preprocessing teks, prediksi sentimen, visualisasi data, serta dashboard interaktif untuk analisis ulasan pelanggan.
